# Лабораторная работа №4. Казино "Гуси и Игроки"
## Введение
Проект представляет собой симулятор казино с необычными участниками - игроками и гусями. В этой игре гуси не просто 
декоративные элементы, а активные участники с уникальными способностями: они могут атаковать игроков, воровать деньги, 
издавать специальные крики и объединяться в стаи. Игроки же делают ставки, паникуют и пытаются выжить в этом 
хаотичном казино.

## Описание файлов
[main.py](src/main.py) - 
точка входа в программу. Содержит функцию main(), которая запускает несколько симуляций с разными параметрами:

1. Симуляция с параметрами по умолчанию
2. Симуляция с 10 шагами
3. Симуляция с фиксированным seed
4. Симуляция с фиксированным seed и 10 шагами

[simulation.py](src/simulation.py) - 
основной модуль симуляции. Содержит функцию run_sim(), которая:

1. Создает объект казино
2. Регистрирует игроков и гусей
3. Выполняет заданное количество шагов симуляции
4. Выводит статистику и результаты

[Wanna_play_kazik.py](src/Wanna_play_kazik.py) - 
Содержит основные классы для работы казино:

- Chip - игровая фишка с номиналом
- CasinoBalance - управление балансами с логами изменений
- Casino - основной класс казино, управляющий всеми событиями

[Players.py](src/Players.py) - 
Содержит классы для работы с игроками:

- Player - базовый класс игрока с именем и балансом
- PlayerCollection - коллекция игроков с дополнительными методами

[We_need_one_more_goose.py](src/We_need_one_more_goose.py) - 
Содержит классы для работы с гусями:

- Goose - базовый класс гуся
- WarGoose - военный гусь, умеющий атаковать игроков
- HonkGoose - гусь-крикун с особыми способностями
- GooseFlock - стая гусей

## Основные возможности
#### События в казино: 
1. Игрок делает ставку - с шансом 33% утраивает ставку
2. Гусь атакует - военный гусь уменьшает баланс игрока
3. Гусь гудет - обычные гуси гудят, супер-гуси используют особые способности
4. Гусь ворует - гусь крадет деньги у игрока
5. Игрок паникует - игрок теряет все деньги
6. Создается фишка - создаются игровые фишки, которые могут объединяться
7. Гуси объединяются - гуси образуют стаю

#### Типы гусей
- Обычный гусь - может только гудеть
  - Военный гусь - может атаковать игроков
  - Гусь-крикун - может использовать особый крик, который приносит деньги игрокам

## Статистика
После симуляции выводится:

- Финальные балансы всех участников
- Доходы гусей
- Количество игроков с деньгами
- Самый богатый игрок
- Самый успешный гусь

## Архитектурные особенности
- Использование MutableMapping и MutableSequence - для создания специализированных коллекций
- Наследование и полиморфизм - разные типы гусей наследуются от базового класса
- Специальные методы Python - использование __add__, __repr__, __call__ и других
- Логирование изменений - CasinoBalance автоматически ведет лог всех изменений балансов
- Вероятностные события - случайный выбор событий с разными весами
